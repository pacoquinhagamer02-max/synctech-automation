Conectar Make / n8n
O webhook é o "canal" por onde o agendamento envia os dados para o Make.com ou n8n, que vai notificar o cliente no WhatsApp e salvar na planilha.
⚡ Como criar o webhook no Make (gratuito)
  1. Acesse make.com e crie uma conta grátis
  2. Clique em Create a new scenario
  3. Adicione o módulo Webhooks → Custom webhook
  4. Clique em Add → copie a URL gerada
  5. Cole abaixo e clique em Testar
  6. Após o teste, adicione o módulo WhatsApp Business para enviar a confirmação
Deixe em branco por enquanto se ainda não tiver — pode configurar depois no Dashboard → Configurações
